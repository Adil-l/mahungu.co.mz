# Mahungu Studio 2.0

Plataforma criativa para geração e gestão de flyers digitais.

## 🚀 Estrutura do Projeto

- `index.html`: Ponto de entrada da aplicação.
- `assets/`:
    - `css/`: Estilos visuais.
    - `js/`: Lógica da aplicação (modularizada).
    - `img/`:
        - `system/`: Ativos fixos da interface.
        - `photos/`: Fotos base e modelos.
- `database/`: Scripts e documentação da persistência de dados.
- `setup/`: Guias de instalação e gerenciamento de dependências.

## 🛠️ Tecnologias
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+ Modules)
- IndexedDB (para armazenamento local persistente)
- Lucide Icons
- html2canvas (para exportação de imagens)

## 📖 Como usar
1. Abra o `index.html` em um navegador moderno (ou use o servidor local).
2. No Painel de Edição, personalize o texto e a imagem.
3. Salve no Histórico ou exporte como PNG.
